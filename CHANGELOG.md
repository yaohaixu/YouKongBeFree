# Changelog

所有重要变更都会记录在此文件中。格式参考 [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)，版本号遵循语义化版本思路。

## [0.4.3] - 2026-07-10

### Added

- 新增安全响应头：CSP、`X-Frame-Options`、`X-Content-Type-Options`、`Referrer-Policy`、`Permissions-Policy`，HTTPS 环境返回 HSTS。
- 新增所有 HTML 页面的 `Content-Security-Policy` meta 和 referrer meta，补齐 CloudBase Hosting 静态页的基础浏览器侧约束。
- 新增非 GET API 请求安全校验头要求，前端统一发送 `X-Requested-With: XMLHttpRequest`。
- 新增登录、报名和普通写操作的内存级限流。
- 新增 `docs/security.md`，记录当前安全控制和遗留风险。
- 新增 `CORS_ORIGINS` 和 `SESSION_MAX_AGE_DAYS` 环境变量说明。

### Changed

- Session 服务端存储从明文 token 调整为 token 哈希，并增加过期时间。
- 登录 token 随机强度提升到 32 字节；新生成 ID 的随机后缀提升到 8 字节。
- CORS 允许头从回显请求头调整为固定白名单。
- 公开协作员接口和登录态接口不再返回手机号，管理员成员管理接口仍返回手机号。
- 上传封面限制为 JPG、PNG、WebP、GIF，单文件最大 6MB。
- 手机号、昵称、模块、活动标题、地点、描述和审核说明增加格式与长度校验。
- 静态资源版本参数升级为 `v=0.4.3`。

### Fixed

- 修复上传错误和请求体过大时可能返回泛化 500 的问题，改为明确 4xx 错误。
- 降低跨站表单提交、点击劫持、内容嗅探和过度暴露成员手机号的风险。

### Removed

- 无。

## [0.4.2] - 2026-07-10

### Changed

- 静态资源版本参数升级为 `v=0.4.2`，避免线上缓存旧 CSS。

### Fixed

- 修复部分页面仍沿用旧黑白重视觉导致风格不统一的问题。
- 优化 PC 和移动端页面层次，让主要内容在浅色系统风格下更清爽、可读。
- 优化滚动进入动效，避免未触发动画时内容区域出现空白。

### Removed

- 无。

## [0.4.1] - 2026-07-09

### Added

- 新增全站保存/审核/撤回等管理操作的“保存成功”轻提示。
- 新增删除报名记录、删除成员、删除活动模块后的确认弹窗与删除反馈。
- 新增动态渲染活动列表、待办任务、报名表和成功页内容的进入动效。
- 审核待办详情新增活动封面图展示。

### Changed

- 普通成员不再展示无意义的待办任务模块，只有协作员能看到自己的审核待办。
- 草稿和审核中活动详情页不再展示报名表，改为说明当前状态暂不开放报名。
- 满员或已结束活动不接受新报名，但已报名手机号仍可找回报名确认页。
- 全站内容最大宽度调整为约 `1200px`，并增加 PC / 移动端自适应左右留白。
- 表单输入框、按钮、管理行、审核卡片和报名表的间距与卡片层次统一优化。
- 静态资源版本参数升级为 `v=0.4.1`，避免线上缓存旧 CSS / JS。

### Fixed

- 修复成员角色仅为成员时仍看到待办任务区的问题。
- 修复审核待办无法查看上传封面图的问题。
- 修复草稿和审核中活动可能在详情页露出报名表的问题。
- 修复部分表单按钮与输入框距离过近、移动端按钮挤压和列表贴边导致的排版问题。

### Removed

- 无。

## [0.4.0] - 2026-07-09

### Added

- 新增成员/协作员多角色模型，YKadmin 仍为系统唯一管理员。
- 新增活动双岗审核流：管理员审核后进入协作员审核，协作员通过后活动公开发布。
- 新增审核意见：通过、退回、拒绝；退回回到发起人，拒绝后不可编辑。
- 新增活动状态：草稿、审核中、退回、拒绝、活动发布、活动人满、活动取消、活动结束。
- 新增活动草稿保存、选择协作员、发起人撤回活动、管理员全部活动查看。
- 新增管理员和协作员待办任务区。
- 新增重复报名识别：同手机号再次报名会直接返回已有报名确认信息。
- 新增报名成功页取消报名功能。
- 新增移动端登录 token 兜底，改善跨域 Cookie 导致的移动端跳转后未登录问题。

### Changed

- 活动创建从“直接发布”调整为“草稿/提交审核”。
- 管理员添加成员时删除“管理员”选项，只保留成员和协作员多选。
- 公开活动列表只展示已发布、已满员和已结束活动。
- 全站静态资源版本参数升级为 `v=0.4.0`。

### Fixed

- 优化顶部导航登录态缓存，减少切换页面时“昵称 · 退出”闪烁。
- 修复移动端输入管理员手机号后可能未成功进入后台的问题。

### Removed

- 移除后台成员角色下拉框中的“管理员”选项，系统只保留默认 YKadmin 管理员。

## [0.3.0] - 2026-07-09

- 新增腾讯云 CloudBase 动态部署能力。
- 新增 CloudBase Hosting 静态站点部署脚本。
- 新增 CloudBase 云函数 `youkongApi` 部署脚本，通过 HTTP 访问服务 `/api` 提供动态接口。

- 默认网站管理员调整为昵称 `有空管理员`、手机号 `13377779999`。

- 修复 CloudBase HTTP 访问服务会剥离 `/api` 前缀导致 Express 路由 404 的问题。
- 修复部署脚本误用 `--httpFn` 导致函数类型不匹配的问题。
